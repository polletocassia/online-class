# User Stories

## Como professor, quero criar uma turma, para organizar meus alunos
- O professor deve estar logado.
- Deve informar o nome da turma.
- Apenas professores podem criar turmas.
- A turma deve aparecer na lista de turmas.

## Como professor, quero publicar atividades, para avaliar meus alunos.
- O professor deve selecionar uma turma.
- Deve informar título e descrição da atividade.
- Deve definir uma data de entrega.
- A atividade deve ficar visível para os alunos da turma.

## Como aluno, quero entregar atividades online para receber avaliação.
- O aluno deve estar matriculado na turma.
- Deve conseguir enviar arquivos.
- O sistema deve registrar a entrega.
- O professor deve visualizar o envio.

## Como aluno, quero visualizar meu calendário para acompanhar os prazos
-  O calendário deve exibir provas e atividades.
- Deve mostrar datas de entrega.
- Apenas eventos da turma do aluno devem aparecer.
- As informações devem ser atualizadas automaticamente.

## Como professor, quero visualizar relatórios gerados por IA, para identificar dificuldades da turma.
- O relatório deve apresentar o desempenho da turma.
- Deve destacar dificuldades dos alunos.
- Apenas professores podem acessar os relatórios.
- O relatório deve estar vinculado a uma turma.

## Como aluno, quero ganhar pontos e medalhas para acompanhar meu desempenho.
- O sistema deve atribuir pontos por atividades.
- O aluno deve visualizar sua pontuação.
- O aluno deve visualizar medalhas conquistadas.
- A pontuação deve ser atualizada automaticamente.
